# UI/UX 디자인 가이드

## 디자인 원칙

### 1. 사용자 중심 (User-Centric)
- 학습에 집중할 수 있는 깔끔한 인터페이스
- 명확한 정보 구조와 네비게이션
- 직관적인 사용자 흐름

### 2. 접근성 (Accessibility)
- WCAG 2.1 AA 준수
- 키보드 네비게이션 지원
- 스크린 리더 호환
- 적절한 색상 대비

### 3. 반응형 (Responsive)
- 모바일 우선 설계
- 태블릿, 데스크톱 최적화
- 유연한 그리드 시스템

### 4. 일관성 (Consistency)
- 통일된 컴포넌트 사용
- 일관된 색상 및 타이포그래피
- 예측 가능한 인터랙션

## 주요 컴포넌트

### 1. 강의 카드 (Course Card)
- 썸네일 (16:9 비율)
- 카테고리 뱃지
- 강의 제목
- 강사명
- 평점 및 수강생 수
- 가격
- 호버 시 미리보기 정보 표시

### 2. 비디오 플레이어
- 재생/일시정지
- 진행 바 (10초 단위 썸네일 미리보기)
- 볼륨 조절
- 배속 조절 (0.5x ~ 2.0x)
- 전체화면
- 자막 (선택사항)
- 다음 강의 자동 재생
- 단축키 지원

### 3. 네비게이션 바
- 로고
- 검색 바
- 카테고리 메뉴
- 사용자 메뉴
  - 내 강의실
  - 위시리스트
  - 알림
  - 프로필

### 4. 진도율 표시
- 프로그레스 바
- 퍼센티지 (%)
- 완료한 강의 수 / 전체 강의 수

### 5. 수강평 카드
- 사용자 아바타 및 이름
- 별점 (1-5)
- 작성일
- 리뷰 내용
- 도움됨 버튼
- 강사 답글

## 인터랙션

### 애니메이션
- 페이지 전환: Fade in/out
- 버튼 호버: Scale + 색상 변화
- 카드 호버: 그림자 증가
- 로딩: Skeleton UI 또는 Spinner

### 피드백
- 버튼 클릭: 색상 변화 + Ripple 효과
- 폼 제출: 로딩 인디케이터
- 성공/에러: Toast 알림
- 진행 중: Progress indicator

### 마이크로 인터랙션
- 좋아요 버튼: 하트 애니메이션
- 북마크: 즐겨찾기 애니메이션
- 체크박스: 체크 애니메이션
