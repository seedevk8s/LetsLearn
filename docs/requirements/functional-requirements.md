# 기능 요구사항

## 1. 사용자 관리

### 1.1 회원가입 및 인증
- [ ] 이메일 회원가입
- [ ] 소셜 로그인 (Google, GitHub, Kakao, Naver)
- [ ] 이메일 인증
- [ ] 비밀번호 찾기/재설정
- [ ] 프로필 관리

### 1.2 사용자 역할
- [ ] 학생 (Student)
- [ ] 강사 (Instructor)
- [ ] 관리자 (Admin)

## 2. 강의 관리

### 2.1 강의 등록 (강사)
- [ ] 강의 기본 정보 입력 (제목, 설명, 카테고리)
- [ ] 커리큘럼 구성 (섹션, 레슨)
- [ ] 동영상 업로드
- [ ] 강의 자료 첨부 (PDF, 코드 파일 등)
- [ ] 가격 설정
- [ ] 미리보기 영상 설정
- [ ] 강의 공개/비공개 설정

### 2.2 강의 검색 및 탐색 (학생)
- [ ] 카테고리별 탐색
- [ ] 키워드 검색
- [ ] 필터링 (가격, 난이도, 평점, 최신순)
- [ ] 추천 강의
- [ ] 인기 강의 랭킹

### 2.3 강의 상세 페이지
- [ ] 강의 소개 및 커리큘럼
- [ ] 강사 정보
- [ ] 수강평 및 평점
- [ ] Q&A 게시판
- [ ] 미리보기 영상 재생

## 3. 수강 관리

### 3.1 강의 구매
- [ ] 장바구니
- [ ] 위시리스트
- [ ] 결제 시스템 (신용카드, 계좌이체, 간편결제)
- [ ] 쿠폰 및 할인 코드
- [ ] 구매 내역 관리

### 3.2 강의 수강
- [ ] 동영상 플레이어 (재생, 일시정지, 배속 조절)
- [ ] 진도율 자동 저장
- [ ] 북마크 기능
- [ ] 메모 기능
- [ ] 강의 자료 다운로드
- [ ] 이어보기 기능

### 3.3 학습 관리
- [ ] 내 강의실 대시보드
- [ ] 학습 진행률 확인
- [ ] 수료증 발급
- [ ] 학습 통계

## 4. 커뮤니티 기능

### 4.1 질문 & 답변
- [ ] 강의별 Q&A 게시판
- [ ] 질문 작성 (코드 하이라이팅 지원)
- [ ] 답변 작성
- [ ] 좋아요/베스트 답변

### 4.2 수강평

- [ ] 좋아요/도움됨 표시
- [ ] 강사 답글

### 4.3 공지사항
- [ ] 강의별 공지사항
- [ ] 업데이트 알림

## 5. 강사 기능
